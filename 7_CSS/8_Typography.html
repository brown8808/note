Serif 글꼴은 각 문자 끝에 추가 세부 정보가 있습니다. Sans-Serif 글꼴은 그렇지 않습니다. 대체 글꼴은 사용자 컴퓨터에 특정 글꼴이 설치되어 있지 않을 때 사용됩니다. word-spacing 속성은 개별 단어의 간격을 변경합니다. letter-spacing 속성은 개별 문자의 간격을 변경합니다. text-align 속성은 텍스트의 가로 정렬을 변경합니다. Google 글꼴은 태그 또는 @font-face 속성이 있는 HTML 파일에서 사용할 수 있는 무료 글꼴을 제공합니다. 로컬 글꼴은 @font-face 속성과 글꼴 소스 경로를 사용하여 문서에 추가할 수 있습니다.
