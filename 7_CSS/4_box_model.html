상자 모델은 웹 페이지에서 공간을 차지하는 요소의 일부를 정의하는 속성 집합 width및 height: 콘텐츠 영역의 너비와 높이입니다. padding: 콘텐츠 영역과 테두리 사이의 간격입니다. border: 콘텐츠 영역과 패딩을 둘러싸는 테두리의 두께와 스타일입니다. margin: 테두리와 요소의 바깥쪽 가장자리 사이의 공간입니다. padding-top padding-right padding-bottom padding-left 한 면씩 테두리 설정 가능 여백 margin-top margin-right margin-bottom margin-left div.headline { width: 400px; margin: 0 auto; } 자동으로 센터에 맞춤 최소 및 최대 높이와 ​​너비 min-width- 이 속성은 요소 상자의 최소 너비를 보장합니다. max-width- 이 속성은 요소 상자의 최대 너비를 보장합니다. p { min-width: 300px; max-width: 600px; } min-height— 이 속성은 요소 상자의 최소 높이를 보장합니다. max-height— 이 속성은 요소 상자의 최대 높이를 보장합니다. p { min-height: 150px; max-height: 300px; } overflow 속성은 상자 외부에서 내용물이 쏟아지거나 넘치면 어떻게 되는지 제어합니다. 가장 일반적으로 사용되는 값은 다음과 같습니다. hidden—이 값으로 설정하면 오버플로되는 모든 콘텐츠가 보기에서 숨겨집니다. scroll- 이 값으로 설정하면 나머지 콘텐츠를 스크롤하여 볼 수 있도록 요소의 상자에 스크롤바가 추가됩니다. visible- 이 값으로 설정하면 오버플로 콘텐츠가 포함 요소 외부에 표시됩니다. 참고로 이것은 기본값입니다. 많은 개발자가 깨끗한 슬레이트로 작업할 수 있도록 이러한 기본값을 재설정하도록 선택합니다. / 초기화 * { margin: 0; padding: 0; } 속성을 사용하여 요소를 보기에서 숨길 수 있습니다 visibility. 속성 visibility은 다음 값 중 하나로 설정할 수 있습니다. hidden— 요소를 숨깁니다. visible— 요소를 표시합니다. collapse— 요소를 축소합니다.
