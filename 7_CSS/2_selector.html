CSS는 유형, 클래스, ID 및 속성별로 HTML 요소를 선택할 수 있습니다. 범용 선택기를 사용하여 모든 요소를 ​​선택할 수 있습니다. 요소는 의사 클래스 선택기를 사용하여 다른 상태를 가질 수 있습니다. 하나의 HTML 요소에 여러 CSS 클래스를 적용할 수 있습니다. 클래스는 재사용할 수 있지만 ID는 한 번만 사용할 수 있습니다. ID는 클래스보다 구체적이고 클래스는 유형보다 구체적입니다. 즉, ID는 클래스의 모든 스타일을 재정의하고 클래스는 유형 선택기의 모든 스타일을 재정의합니다. 여러 선택기를 함께 연결하여 요소를 선택할 수 있습니다. 이는 특이성을 높이지만 필요할 수 있습니다. 선택자를 공백으로 구분하여 중첩된 요소를 선택할 수 있습니다. 관련 없는 여러 선택자는 선택기 이름을 쉼표로 구분하여 동일한 스타일을 받을 수 있습니다.
